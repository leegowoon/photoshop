# photoshop

# 화면 크기 조절 🍒
- alt + 마우스휠 위아래로 조절
- ctrl + 0 --> 화면크기에 맞춤
- ctrl + 1 --> 100% 크기로 보임

# 레이어 복사 🐿️
- ctrl + j
- alt + 레이어를 위로 이동 --> 복사된 레이어가 위에 위치
- alt + 레이어를 아래로 이동 --> 복사된 레이어가 아래로 위치

# 이미지 크기 조절
- ctrl + t
- 조절시 alt + 조절 --> 기준을 가운데로 변경

# group 만들기
- ctrl + G --> 만들기
- ctrl + shift + G --> 그룹풀기
